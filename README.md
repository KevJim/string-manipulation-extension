# String Manipulation

Port of JetBrains String Manipulation plugin for VS Code. Provides case switching (camelCase, snake_case, kebab-case, etc.) and text alignment (vertical alignment, columns, justification) utilities.

## Features

### Casing

- To camelCase
- To snake_case
- To kebab-case
- To PascalCase
- To SCREAMING_SNAKE_CASE
- To dot.case
- To Title Case
- To Sentence case
- To UPPER CASE
- To lower case
- Invert Case
- Toggle Case

### Alignment

- Align Carets Vertically
- Align to Columns (Separator)
- Justify Left
- Justify Center
- Justify Right

## Usage

Right-click in the editor and select "String Manipulation" from the context menu, or press `Alt+M` (Windows/Linux) or `Cmd+Alt+M` (Mac) to open the Quick Pick menu.

## Development

```bash
npm install
npm run compile
npm test
```
